== Description ==

1. При активации плагина создается пустая база для todo
2. В админ панели создаются две страницы
В "To Do" вывод всех записей из БД, кнопка для синхронизации API, поле для поиска по Title.
На второй странице выводиться простой список последних 5 задач, в произовльном порядке через шорткода

В шорткоде через параметры можно указать кол. задач, статус и юзер id.
